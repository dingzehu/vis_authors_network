Collaborator Network

Have you ever wondered how much scientific authors collaborate with each other? Further, which author likes to collaborate with which other authors or even institutions?

This website helps you with exactly that, we can visualize an author with their connected collaborators weighted by the number of shared publications. You can even specify which type of graph embedding should be used for the visualization.

Give it a try and search for your favourite author above!
